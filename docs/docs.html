Models

Not all field are gauranteed as they may be subject to sensitivity filters for specific endpoints.

  • User

    Represents a Kakapo platform user.

    {
  "kakapo_id": string,
  "display_name": string,
  "email": string,
  "about": string,
  "rank": int,
  "badge": int,
  "joined": int,
  "last_online": int,
  "pfp": string,
  "socials": Array<string>,
  "muted_until": int,
  "banned_until": int,
  "deleted": boolean,
  "public_id": string
}

  • PostTag

    Represents a tag that is attached to a Kakapo post to provide added description.

    {
  "tag": string,
  "post_id": int
}

  • Post

    Represents a Kakapo post.

    {
  "id": string,
  "author": int,
  "media": string,
  "content": string,
  "sent_at": int
}

  • FriendRequest

    Represents a friend request between two Kakapo users.

    {
  "from": int,
  "to": int,
  "sent_at": int,
  "accepted": boolean
}

  • Comment

    Represents a comment on a Kakapo Post

    {
  "id": string,
  "content": string,
  "created_at": int
}
